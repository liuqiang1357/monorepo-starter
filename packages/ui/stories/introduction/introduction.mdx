# Welcome to UI Component Library

This is the Storybook for our UI component library (`@repo/ui`). It showcases reusable components built with React and styled with Tailwind CSS that are shared across all applications.

## Getting Started

To explore the components:

1. Browse components in the sidebar
2. View examples and variations
3. Test component interactions
4. Copy code snippets for your project

## Components

Browse the sidebar to explore all reusable components with live examples and documentation.

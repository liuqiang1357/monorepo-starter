# UI Component Library

This Storybook contains the UI components from `@repo/ui` package.

## Usage

Navigate through the components in the sidebar to see examples, documentation and interactive demos.

## Available Components

- **Button**: A customizable button component with multiple variants and sizes.

## Development

All components are stored in the `packages/ui` directory. To add a new component:

1. Create the component in `packages/ui/src/components`
2. Add a corresponding story in `apps/storybook/stories`
3. Run `pnpm dev` in the storybook directory to preview your changes
